# Jegerportretter

Legg portrettbilder av jegere i denne mappen.

Appen er satt opp med disse foreslåtte filnavnene:

- `jeger_01.png` – Den stille
- `jeger_02.png` – Den sterke
- `jeger_03.png` – Sporleseren
- `jeger_04.png` – Håndverkeren

Du kan også bruke egne navn, for eksempel:

- `nara.png`
- `ulv.jpg`
- `aila.webp`

Når du lager eller redigerer en jeger i appen, skriver du filbanen slik:

`assets/portraits/nara.png`

Hvis bildet mangler, viser appen `portrait-placeholder.svg`.
